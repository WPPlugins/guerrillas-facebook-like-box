== Description ==

This is a plugin that adds a widget that you can use to show a Facebook like box in the sidebar of your WordPress powered website.

== Changelog ==

= 1.1 =
* Updated widget with many additional style options

== Installation ==

1. Add directly through your WordPress admin panel. Alternatively, you can upload 'guerrilla-facebook-like-box.zip' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the newly created widget in any sidebar you wish through the 'Widgets" menu in WordPress
